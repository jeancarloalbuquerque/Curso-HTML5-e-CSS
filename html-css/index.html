Desafios

  • Redes Sociais

    • Criar uma página sobre com uma foto, breve descrição e links para redes sociais

  • Tags em HTML

    • Criar uma página de referência para elementos HTML. Ao passar o mouse em cima do elemnto desejado, deverá aparecer como descrição a tag utilizada para criar o elemento.

  • Biblioteca de Vídeos

    • Criar um site de biblioteca de vídeos, a página principal terá todos os vídeos, ao clicar em um vídeo o usuário sera redirecionado para a página do vídeo em questão. A baixo do vídeo terá um link para ver a playlist completa no youtube.
